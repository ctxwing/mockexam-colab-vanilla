# 관리자 테이블 레이아웃 가이드

이 문서는 관리자 페이지의 표준 테이블 레이아웃을 정의합니다. 모든 프로젝트에서 공통으로 적용 가능한 UI/UX 패턴입니다.

---

## 1. 전체 페이지 구조

```
┌─────────────────────────────────────────────────────────────────────────┐
│ [헤더] 시스템명 / 버전                              사용자명 | 로그아웃 │
├──────────┬──────────────────────────────────────────────────────────────┤
│          │ [페이지 타이틀 영역]                                         │
│  좌측    │ 페이지 제목 + 설명                        [주요 액션 버튼]   │
│  메뉴    ├──────────────────────────────────────────────────────────────┤
│          │ [통계 카드 영역] - 4개 카드 가로 배치                        │
│  (접힘   ├──────────────────────────────────────────────────────────────┤
│   가능)  │ [검색/필터 영역]                                             │
│          ├──────────────────────────────────────────────────────────────┤
│          │ [테이블 영역]                                                │
│          │ - 테이블 제목 + 총 건수                                      │
│          │ - 컬럼 헤더 (정렬 가능)                                      │
│          │ - 데이터 행                                                  │
│          │ - 페이지네이션                                               │
├──────────┴──────────────────────────────────────────────────────────────┤
│ [푸터] 시스템 상태 표시                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

---

## 2. 좌측 사이드바 메뉴

### 2.1 구성 요소
- 시스템 로고/명칭 (상단)
- 네비게이션 메뉴 항목들
- 각 메뉴: 아이콘 + 텍스트
- 현재 페이지 하이라이트 (배경색 또는 좌측 바)
- 하단: 시스템 상태 표시 (예: "● 시스템 온라인")

### 2.2 접힘 기능
- 접힘 토글 버튼: 헤더 영역 우측에 `<` / `>` 아이콘
- 펼침 상태: 아이콘 + 텍스트 표시 (너비 약 200-250px)
- 접힘 상태: 아이콘만 표시 (너비 약 60-70px)
- 접힘 시 메뉴 hover하면 툴팁으로 메뉴명 표시

### 2.3 메뉴 예시 구조
```
📊 대시보드
👥 [엔티티] 관리        ← 현재 페이지 하이라이트
📋 [관련 기능] 관리
📁 마스터 데이터 관리
📝 로그 관리
⚙️ 관리자 설정
```

## 3. 페이지 타이틀 영역

### 3.1 좌측
- 페이지 제목 (큰 폰트, 볼드)
- 페이지 설명 (작은 폰트, 회색)

### 3.2 우측
- 주요 액션 버튼 (예: "신규 등록", "추가" 등)
- 버튼 스타일: Primary 색상, 아이콘 + 텍스트

```
환자 관리                                              [+ 신규 환자 등록]
환자 정보를 등록하고 관리합니다.
```

---

## 4. 통계 카드 영역

### 4.1 레이아웃
- 4개 카드를 가로로 균등 배치 (반응형: 모바일에서 2x2 또는 1열)
- 각 카드 사이 간격: 16-24px

### 4.2 카드 구성
```
┌─────────────────┐
│ 🔵 [아이콘]     │
│                 │
│ [라벨 텍스트]   │  ← 작은 폰트, 회색
│ [숫자 값]       │  ← 큰 폰트, 볼드, 강조색
└─────────────────┘
```

### 4.3 카드 색상 구분
- 각 카드별로 아이콘 색상을 다르게 (파랑, 초록, 노랑, 보라 등)
- 배경: 흰색 또는 연한 회색
- 테두리: 연한 회색 또는 그림자

### 4.4 예시
| 아이콘 | 라벨 | 값 |
|--------|------|-----|
| 👥 파랑 | 총 [엔티티] 수 | 20 |
| 🟢 초록 | 이번 달 신규 | 0 |
| 🟡 노랑 | 오늘 신규 | 0 |
| 🟣 보라 | [특정 상태] 보유 | 20 |

## 5. 검색/필터 영역

### 5.1 레이아웃
- 카드 형태로 감싸기 (배경 흰색, 테두리 또는 그림자)
- 제목: "[엔티티] 검색" + 부제: "[엔티티] 정보를 검색하세요"

### 5.2 필터 구성 요소 (가로 배치)
```
[🔍 검색어 입력]  [드롭다운 필터]  [날짜 From 📅]  [날짜 To 📅]
```

### 5.3 필터 타입별 UI
- 텍스트 검색: 돋보기 아이콘 + placeholder 텍스트
- 드롭다운: 기본값 "[ALL]" 선택 가능
- 날짜 선택: 캘린더 아이콘 + 날짜 포맷 (YYYY. MM. DD.)

### 5.4 반응형
- 데스크톱: 한 줄에 모든 필터 배치
- 태블릿/모바일: 2열 또는 1열로 스택

## 6. 테이블 영역

### 6.1 테이블 헤더 정보
```
[엔티티] 목록
총 20건의 [엔티티]
```

### 6.2 컬럼 헤더

#### 정렬 기능
- 정렬 가능 컬럼: 컬럼명 옆에 정렬 아이콘 표시
- 아이콘 상태:
  - 기본: `↕` (위아래 화살표, 연한 색)
  - 오름차순 활성: `↑` (위 화살표, 진한 색)
  - 내림차순 활성: `↓` (아래 화살표, 진한 색)
- 첫 번째 컬럼은 기본 오름차순 정렬 적용

#### 컬럼 구성 예시
```
| [ID] ↑ | [이름] ↕ | [날짜1] ↕ | [상태] ↕ | [날짜2] ↕ | [날짜3] ↕ | [메모] | [작업] |
```

### 6.3 데이터 행

#### 셀 타입별 표시
- ID: 고정 너비, 모노스페이스 폰트 권장
- 텍스트: 좌측 정렬, 긴 텍스트는 말줄임(...) 처리
- 날짜: YYYY-MM-DD 형식, 중앙 또는 좌측 정렬
- 상태 뱃지: 원형 또는 라운드 뱃지, 색상으로 구분
  - 예: 🔵 M (남성/Male), 🔴 F (여성/Female)
- 민감 정보: 마스킹 처리 (예: 김**, 1990-**-**)

#### 작업 컬럼
- 우측 끝에 배치
- 아이콘 버튼으로 표시: [✏️ 수정] [🗑️ 삭제]
- 버튼 간격: 8px
- hover 시 배경색 변경

### 6.4 행 스타일
- 짝수/홀수 행 배경색 구분 (선택사항)
- hover 시 행 전체 배경색 변경
- 선택된 행 하이라이트 (체크박스 있는 경우)

## 7. 페이지네이션

### 7.1 레이아웃
```
현 20건 중 1-10 표시                    페이지 당 행 [10 ▼]  [이전] [다음]
```

### 7.2 구성 요소
- 좌측: 현재 표시 정보 (예: "현 20건 중 1-10 표시")
- 우측:
  - 페이지 당 행 수 선택 드롭다운 (10, 20, 50, 100)
  - 이전/다음 버튼
  - (선택) 페이지 번호 직접 선택

### 7.3 버튼 상태
- 첫 페이지: "이전" 버튼 비활성화
- 마지막 페이지: "다음" 버튼 비활성화

## 10. 월별 통계 차트 영역

### 10.1 위치
- 통계 카드 영역 바로 아래
- 관리 메뉴 또는 테이블 영역 위

### 10.2 차트 구성
```
┌─────────────────────────────────────────────────────────────────────────┐
│ 월별 통계 (최근 12개월)                                                 │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│   ▓▓                                                          ──●──    │
│   ▓▓  ▓▓                                              ──●──            │
│   ▓▓  ▓▓  ▓▓                                  ──●──                    │
│   ▓▓  ▓▓  ▓▓  ▓▓                      ──●──                            │
│   ▓▓  ▓▓  ▓▓  ▓▓  ▓▓          ──●──                                    │
│   ▓▓  ▓▓  ▓▓  ▓▓  ▓▓  ▓▓  ●──                                          │
│  ─────────────────────────────────────────────────────────────────     │
│   1월  2월  3월  4월  5월  6월  7월  8월  9월 10월 11월 12월            │
│                                                                         │
│   ■ 티켓 수    ■ 답변 수    ● 다운로드 수 (추세선)                      │
└─────────────────────────────────────────────────────────────────────────┘
```

### 10.3 차트 타입
- 복합 차트 (ComposedChart): 막대 + 선 그래프 조합
- 막대 그래프: 월별 티켓 수, 월별 답변 수
- 선 그래프 (추세선): 월별 다운로드 수

### 10.4 데이터 구조
```typescript
interface MonthlyData {
  month: string;        // "2025-01"
  monthLabel: string;   // "1월"
  ticketCount: number;  // 해당 월 티켓 수
  replyCount: number;   // 해당 월 답변 수
  downloadCount: number; // 해당 월 다운로드 수
}
```

### 10.5 색상
- 티켓 수 막대: 파란색 (#3B82F6, blue-500)
- 답변 수 막대: 초록색 (#22C55E, green-500)
- 다운로드 추세선: 보라색 (#A855F7, purple-500)

### 10.6 기능
- X축: 월 라벨 (1월 ~ 12월)
- Y축 좌측: 티켓/답변 수
- Y축 우측: 다운로드 수
- 툴팁: 마우스 호버 시 상세 수치 표시
- 범례: 하단에 각 데이터 항목 표시

### 10.7 라이브러리
- recharts 사용 권장
- 설치: `npm install recharts`

### 10.8 반응형
- 데스크톱: 전체 너비, 높이 350px
- 태블릿: 전체 너비, 높이 300px
- 모바일: 전체 너비, 높이 250px, 범례 세로 배치
