[프로젝트 개요]
AICE 자격증 가상 모의고사 웹 서비스 개발

[프로젝트 배경 및 목표]
- AICE 자격증 독자 및 수강생들에게 추가적인 학습 콘텐츠(모의고사, 해설)를 제공하되, 서버 운영 및 유지보수 비용을 최소화하는 것이 목표입니다. 이를 위해 서버 및 DB가 없는 정적웹사이트 구축을 진행하고자 합니다. (AICE 자격증 독자 및 수강생 대상의 교육용 모의고사 허브 구축)
- 상세 기획이 완료된 상태로, 저비용 고효율 운영에 최적화된 깔끔한 웹 서비스를 신속하게 구축하는 것을 목표로 합니다.

[과업 범위]
1. 수행 범위
- UI/UX 디자인: 첨부된 디자인과 유사한 형태로 진행
- 프런트엔드/Client 개발: HTML, CSS, Vanilla JS를 이용한 정적 웹사이트 전체 개발
- 배포: GitHub Pages를 통한 최종 웹사이트 배포 및 커스텀 도메인 연결

2. 상세 기능 요구 사항
2-1. 사용자 이용 흐름 (User Flow)
- 모바일(QR 접속): 서비스 소개 및 "실습을 위해 PC로 접속하세요" 안내 → [링크 복사] 및 [카톡 공유] 기능 제공
- PC 접속: 도서 인증 페이지 노출 → 도서 내 난수(OTP) 입력 → 인증 성공
- 로비 진입: 회차별(1~N회) 학습 리스트 노출 (Local Storage로 인증 상태 유지)
- 학습 실행: Google Colab 실습 연결 / 웹페이지 내 정답 및 해설 출력 / 외부 강의 링크 연결
2-2. 주요 개발 범위 및 기능
- UI/UX 디자인: 반응형 웹 적용 (모바일 안내/PC 인증/로비/해설 뷰어 등 총 4개 주요 화면)
- 클라이언트 인증: Local Storage 기반의 간단 인증 로직 (서버 통신 없음)
- 데이터 동적 로딩: 회차 정보, 링크, 해설 텍스트 등을 JSON 파일로 관리하여 유지보수성 확보
- 외부 연동: * Google Programmable Search Engine 연동 (라이브러리 공식 문서 검색용)
- Google Colab 및 외부 유료 강의 플랫폼(아웃링크) 연결
- 콘텐츠 보호: 프론트엔드 레벨의 우클릭/드래그 방지 적용

[지원 디바이스 및 디자인]
1. 지원 디바이스
- PC 웹 브라우저
- 모바일 웹 브라우저 (안내 페이지)

[기술 스택]
- 언어: HTML5, CSS3, JavaScript (Vanilla JS 선호, React/Vue 등 프레임워크 사용 시 라이브러리 최소화)
- 배포: GitHub Pages 배포 및 도메인 연결 설정 경험 필수
- 구조: 데이터와 로직이 분리된 유지보수가 용이한 코드 구조

[클라이언트 준비 사항]
1. 문서 및 자료
- 상세 기능 정의 및 화면 흐름 기획 문서 (즉시 착수 가능 수준)
- 회차별 문제/정답/해설 데이터, Colab 링크, 강의 URL
- 인증에 사용할 난수 목록
- 서비스 로고 및 가이드 문구
2. 투입 인력 및 조직
- 프로젝트 관련 의사결정 및 커뮤니케이션을 위한 담당자
3. 계정 및 기술 환경
- GitHub Repository 및 배포에 사용할 도메인

[주요 일정]
1. 희망 착수일: 계약 후 즉시
2. 최종 오픈 희망일: 협의 가능

[지원 자격 및 우대 사항]
1. 지원 자격
- HTML, CSS, JavaScript(ES6+)를 이용한 프론트엔드 개발에 능숙한 개발자
- GitHub Pages를 이용한 정적 웹사이트 배포 및 커스텀 도메인 연결 경험
2. 우대 사항
- React/Vue 등 프레임워크 없이 순수 JavaScript(Vanilla JS)로 인터랙티브 웹을 구현해 본 경험
- 교육용 서비스 또는 유사한 콘텐츠 기반 웹사이트 개발 경험
- 데이터와 로직을 분리하여 유지보수성이 높은 코드 구조를 설계할 수 있는 분

[산출물]
- 웹사이트 전체 소스 코드 일체
- GitHub Pages에 최종 배포 완료된 웹사이트
- 향후 회차 추가 등 데이터(JSON) 수정 방법에 대한 간단한 매뉴얼

[계약 관련 특이 사항]
- 백엔드 구축 없이 프론트엔드 작업에 집중할 수 있는 깔끔한 프로젝트입니다.
- 특히 교육용 서비스에 대한 이해도가 높고, GitHub Pages를 활용한 정적 웹 배포 경험이 풍부한 파트너분을 찾습니다.
