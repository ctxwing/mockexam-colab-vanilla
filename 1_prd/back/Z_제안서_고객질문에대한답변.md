# 고객 질문에 대한 답변

## 질문1) 간단한 인증 방식 구현 경험 또는 구현 방안

**프로토타입에서 실제 구현 및 검증 완료**

- **구현 방식**: 16자리 코드 입력 → Next.js API 서버 검증 → 쿠키 세션 발급(1일) → 대시보드 자동 이동

- **코드 관리**: Google Sheets 저장으로 별도 DB 없이 관리자가 시트에서 직접 추가/삭제

- **보안 적용**: ① 비인가 접근 자동 리다이렉트 ② Open Redirect 차단 ③ 쿠키 보안(HttpOnly, Secure, SameSite)

- **편의성**: 대소문자 무관 인증으로 입력 오류 최소화

- **검증**: 프로덕션 환경 18개 통합 테스트 100% 통과, 데모 사이트 https://qr-mockexam-colab-nextjs.vercel.app 에서 확인 가능
