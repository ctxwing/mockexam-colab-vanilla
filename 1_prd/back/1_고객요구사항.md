## 프로젝트 개요
Google Colab 기반 교육용 가상 모의고사 웹 시스템 구축.

## 프로젝트 배경 및 목표
- IT 수험서(AICE, 빅데이터 분석기사) 독자들이 책에 있는 QR코드를 통해 손쉽게 실습 환경(Google Colab)에 접속하여 모의고사를 풀 수 있도록 돕는 '브릿지(Bridge) 웹사이트' 구축.
- 복잡한 회원가입이나 서버 구축 없이, 도서 구매 인증(코드 입력)만으로 실습이 가능한 간편한 시스템 지향.

## 과업 범위

### 1. 프론트엔드 개발 (반응형 웹)
- **모바일 접속 페이지**: QR로 접속 시 모바일임을 감지하고, PC 접속을 유도하는 안내 페이지 (URL 복사 기능 포함).
- **인증 페이지 (PC)**: 도서 내 인증 코드(난수) 입력 및 검증 UI.
- **모의고사 리스트**: 회차별 모의고사 선택 및 데이터 다운로드 안내 UI.
- **결과 페이지**: 구글 폼(제출) 링크 및 정답/해설 보기 버튼 제공.

### 2. 기능 및 로직 구현
- **간편 인증**: DB 없이 소스 코드 내(또는 환경변수) 저장된 20개의 인증 코드와 사용자 입력값 비교 (대소문자 무시).
- **Colab 연동**: 버튼 클릭 시 미리 준비된 Google Colab 노트북 URL로 새 창 연결.
- **데이터 제공**: pandas에서 바로 로드할 수 있는 CSV 파일 URL(GitHub Raw 등) 제공 및 클립보드 복사 기능.

### 3. 배포 및 인프라
- **Serverless/Static Hosting**: Vercel, Netlify, GitHub Pages 등 무료 또는 저비용 호스팅 환경 구축.
- 별도의 백엔드 서버나 DB 구축 없음.

## 기술 스택

- **Hosting**: Vercel (권장)
- **Language**: TypeScript / JavaScript
