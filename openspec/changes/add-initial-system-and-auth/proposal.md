# 변경 제안: 초기 시스템 구축 및 인증 기능 구현

## Why
교육용 가상 모의고사 시스템의 기반을 마련하기 위함입니다. 사용자가 QR 코드를 통해 접속했을 때 적절한 안내를 받고, 도서에 포함된 인증 코드를 입력하여 모의고사 실습 환경(Google Colab)에 안전하게 접근할 수 있는 핵심 기능을 조기에 구현합니다.

## What Changes
- **초기 프로젝트 환경 설정**: Next.js 16+ (React 19), TypeScript, Tailwind CSS, shadcn/ui 기반의 프로젝트 구조를 생성합니다.
- **반응형 랜딩 페이지**: 모바일 접속 시 PC 접속을 유도하는 안내 기능을 포함한 랜딩 페이지를 구현합니다.
- **인증 시스템 엔진**: 도서 내 인쇄된 20개의 난수 코드를 서버 측 상수로 관리하고, 사용자 입력값과 대소문자 무시 조건으로 비교하는 DB-less 인증 로직을 구현합니다.
- **모의고사 대시보드 및 결과 페이지**: 인증 성공 후 이동하는 대시보드 페이지에서 Google Colab 링크를 제공하며, 문제 풀이 후 구글 폼 기반의 답안 제출 및 정답/해설 확인 기능을 구현합니다. 모든 데이터는 정적 JSON 또는 상수로 관리됩니다.

## Impact
- **영향을 받는 가공 역량(Capabilities)**: `landing-page`, `auth-system`, `dashboard`
- **핵심 파일**: `app/page.tsx`, `app/auth/page.tsx`, `components/auth-form.tsx`, `lib/auth-codes.ts`
- **보안**: `3_prj_docs/NextJS_React_보안취약점패치적용_보고_참고.md`에 명시된 보안 패치를 준수하며, 최신 버전의 프레임워크를 적용하여 CVE 이슈를 방지합니다.
