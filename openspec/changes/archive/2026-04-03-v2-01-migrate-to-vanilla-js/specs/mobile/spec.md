## ADDED Requirements
### Requirement: 모바일 안내 페이지
모바일에서 접속 시 PC 접속을 안내하는 페이지를 표시해야 한다(SHALL). 상단에 AICE 로고, 중앙에 도서 표지 이미지와 안내 문구가 노출된다.

#### Scenario: 모바일 접속 안내
- **WHEN** 사용자가 모바일로 접속한다
- **THEN** 상단에 AICE 로고, 중앙에 도서 표지 이미지와 "PC 실습 환경에 최적화된 서비스입니다" 안내가 표시된다

#### Scenario: 링크 공유 기능
- **WHEN** 사용자가 링크 복사 또는 카톡 공유를 클릭한다
- **THEN** 해당 기능이 실행된다

### Requirement: URL 복사 기능
사용자는 현재 페이지 URL을 클립보드에 복사할 수 있어야 한다(SHALL).

#### Scenario: URL 복사
- **WHEN** 사용자가 "URL 복사" 버튼을 클릭한다
- **THEN** 현재 페이지 URL이 클립보드에 복사되고 성공 메시지가 표시된다

### Requirement: 카카오톡 공유 기능
사용자는 카카오톡으로 현재 페이지 링크를 공유할 수 있어야 한다(SHALL).

#### Scenario: 카카오톡 공유
- **WHEN** 사용자가 "카카오톡으로 링크 보내기" 버튼을 클릭한다
- **THEN** 카카오톡 공유 다이얼로그가 열린다

### Requirement: 반응형 디바이스 감지
시스템은 사용자 에이전트를 통해 모바일/PC를 자동으로 감지해야 한다(SHALL).

#### Scenario: 모바일 디바이스 감지
- **WHEN** 모바일 기기로 접속한다
- **THEN** 모바일 안내 페이지가 표시된다

#### Scenario: PC 디바이스 감지
- **WHEN** PC 브라우저로 접속한다
- **THEN** 인증 페이지가 표시된다
