□ 프로젝트 제안 개요
- 귀사의 요구사항(AICE, 빅데이터 분석기사 독자 대상 브릿지 사이트)를 면밀히 분석하였습니다.
- 유사 구축 경험으로 별도의 DB 없이도  자유롭게 운영할 수 있도록 구글 드라이브와 구글 시트를 통합하여 구성하였습니다.
- 이로인해 관리자가 사용이 편리한 구글 생태계에서 용이하게 운영할 수 있도록 서버리스 솔루션을 제안하며, 운영비용최소화를 달성했습니다.

□ 제안드리는 핵심 3가지 차별성
① 제로 레이턴시(Zero-Latency) 데이터 동기화 (ISR (증분정적사이트 생성)기술 + 구글드라이브  내 문서와 Google Appscript를 Hybrid 적용)
- Google Sheets를 DB처럼 활용하되 Next.js ISR 기술을 적용하여 1초 이내의 압도적인 로딩 속도를 구현했습니다.
- 일반적인 GAS 연동 방식의 느린 응답 문제를 해결하여 사용자에게 최상의 집중 환경을 제공합니다.

② 관리자가 익숙한 구글 서비스를 사용한 운영 생태계 (Fully Google Integrated)
- 배포된 사이트는 vercel에서 운영됩니다. 
- 관리자 기능은 구글 드라이브, 시트, 문서, 폼을 하나의 폴더로 통합하여 비전문가도 쉽게 데이터를 관리할 수 있습니다.
- 개발자 코드 수정 없이 시트내의 값만 조작만으로 모의고사 회차 추가 및 링크 변경이 가능하여 총운영 비용(TCO)을 줄이는 형태로 구현하였습니다.

③ 사용자 이탈을 방지하는 지능형 학습 최적화 UX (Intelligent Gateway)
- 모바일/PC 환경 자동 감지 및 안내 기능을 통해 사용자의 접근성을 극대화했습니다.
- 구글 폼 제출 시 응시자 인증 코드를 자동 입력(Auto-Prefill)하여 입력 오류와 이탈을 완벽히 방지합니다.

□ 개발 검증 및 산출물
- 저희는 요구사항을 모두 반영한 프로토타입을 완성하였으며, 아래 산출물을 포함합니다.
- 1) Next.js 기반 반응형 웹 소스 2) 운영 관리용 Google Sheets/Google Apps Script 3) 상세 운영 및 환경 구축 가이드 3종

□ 현재 프로토 타입 기능 요약
- 핵심 엔진: Next.js 16+ 기반의 초경량/초고속 서버리스 아키텍처.
- 데이터 연동: Google Sheets API 및 ISR 온디맨드 재검증 기능 구현 완료.
- 사용자 편의: 모바일 감지, 클립보드 복사, 구글 폼 자동 인증 연동.
- 보안 기능: 16자리 유니크 인증 코드 기반 접근 제어 및 보안 미들웨어 적용.
