# Google Colab 기반 교육용 가상 모의고사 웹 시스템 구축 요구사항

## 1. 프로젝트 개요
본 프로젝트는 **AICE 데이터 분석 과정** 학습자를 위한 교육용 가상 모의고사 시스템을 구축하는 것입니다. 학습자가 도서의 QR 코드를 통해 접속하여 간단한 인증 후, Google Colab 환경에서 실습형 모의고사를 진행할 수 있도록 지원합니다.

*   **핵심 도구**: Google Colab (필수)
*   **주요 흐름**: 도서 QR 접속 → 인증 코드 입력 → 모의고사 진행
*   **서비스 성격**: 학습 및 자가진단 목적 (자동 채점 및 순위 기능 제외)

---

## 2. 전체 서비스 프로세스
1.  **QR 스캔**: 도서 내 인쇄된 QR 코드를 스캔합니다.
2.  **안내 페이지(모바일)**: 모바일 전용 안내 페이지에 접속합니다.
3.  **PC 전환**: [PC에서 열기] 링크를 통해 PC 전용 페이지로 이동합니다.
4.  **인증 코드 입력**: 도서 특정 페이지에 명시된 인증 코드(난수)를 입력합니다.
5.  **모의고사 진입**: 인증 통과 후 모의고사 안내 및 실행 페이지로 이동합니다.
6.  **실습 진행**: Google Colab 링크를 실행하여 문제 풀이를 수행합니다.
7.  **제출 및 자가채점**: 답안 제출 후 제공되는 정답/해설을 통해 스스로 채점합니다.

---

## 3. 상세 요구사항

### 3.1 실습 환경
*   **Google Colab 기반**: 별도의 로컬 Jupyter Notebook 설치 없이 브라우저에서 즉시 실행 가능해야 합니다.
*   **웹 기반**: 모든 과정은 웹 브라우저를 통해 이루어집니다.

### 3.2 데이터 제공 방식
*   **URL 기반 데이터 로딩**: 실습 데이터는 외부 URL을 통해 제공됩니다.
*   **Pandas 활용**: `df = pd.read_csv("https://example.com/data.csv")`와 같이 시험 진행 중 즉시 로드가 가능해야 합니다.

### 3.3 인증 시스템
*   **단순 인증**: 회원가입, ID/PW, 관리자 계정이 없는 구조입니다.
*   **인증 코드**: 도서에 인쇄된 20여 개의 난수 코드 중 하나를 입력하여 인증합니다.
*   **비교 로직**: 대소문자 구분 없는 단순 문자열 비교 방식을 사용합니다.
*   **DB 미사용**: 별도의 데이터베이스 구축 없이 서버 측 설정이나 정적 파일 기반으로 검증 가능해야 합니다.

---

## 4. 기능 범위

### ✅ 포함 기능
*   **안내 페이지**: QR 접속 시 최초 노출되는 안내 화면
*   **인증 페이지**: PC 접속 후 인증 코드를 입력받는 화면
*   **모의고사 대시보드**: 모의고사 안내 및 Colab 실행 링크 제공
*   **제출 시스템**: 외부 폼(예: Google Forms) 등을 활용한 제출 기능
*   **정답 및 해설**: 자가채점을 위한 정답지 및 해설 안내

### ❌ 제외 기능
*   자동 채점 시스템 및 실시간 성적 산출
*   순위표(Leaderboard) 및 전체 통계 리포트
*   기존 LMS(학습관리시스템)와의 연동
*   네이티브 모바일 앱 개발
*   구름 IDE와 같은 별도 컨테이너 시험 환경 구축

---

## 5. 개발 원칙 및 방향
*   **단순성**: 복잡한 아키텍처를 지양하고 저비용/저유지보수 구조로 설계합니다.
*   **독립성**: 특정 유료 솔루션에 종속되지 않는 독립적인 시스템을 목표로 합니다.
*   **사용자 경험(UX)**: 교육용 서비스 특성에 맞춰 누구나 직관적으로 이용할 수 있도록 합니다.
*   **확장성**: 향후 요구에 따라 계정 기반 인증으로 확장할 수 있는 유연성을 고려합니다.
