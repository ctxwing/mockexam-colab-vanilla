코랩(Colab) 링크 만들기

단순히 구글 드라이브(GDF)에 파일을 올린다고 해서 바로 코랩(Colab) 링크가 되는 것은 아닙니다. 그건 그냥 '파일 저장'일 뿐입니다.

학생들이 클릭했을 때 바로 Google Colab 정식 편집기가 뜨게 하려면, 아래 방법으로 링크를 만드셔야 합니다.


STEP1 : 가장 직관적인 방법 (실제 실행 주소 복사)
GDF에 올린 파일을 마우스 우 클릭하여 상단의 **[연결 앱] -> [Google Colaboratory]**를 클릭하여 실제 코랩으로 파일을 엽니다.
코랩이 정상적으로 열리면, 브라우저 주소창에 떠 있는 긴 주소를 그대로 복사합니다.
그 주소를 ‘모의고사_대시보드_관리’ 문서 >  ‘exam’시트 > colabUrl 열의 해당 항목에 넣습니다.


STEP2: 
- 위 STEP1 내용을 확인 후,  ‘모의고사_대시보드_관리’ 상단의 💎 [모의고사 통합 관리] 메뉴 중에서 ’3. 수정한 내용 서버 즉시 반영 (Purge) [⚠️ 주의
!]를 실행하면, 웹서비스에 반영됩니다.
