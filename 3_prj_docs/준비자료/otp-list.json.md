# 인증 코드 목록 (OTP List)

## 개요
도서 [페이지 21] 하단에 인쇄된 8자리 난수 코드 목록입니다.
대소문자 구분 없이 입력 가능하며, 하이픈(-) 자동 보정 로직이 적용됩니다.

## 인증 코드 (총 20개)

```json
[
  "AICE2025",
  "AICE2026",
  "AICE2027",
  "AICE2028",
  "AICE2029",
  "BIGD2025",
  "BIGD2026",
  "BIGD2027",
  "BIGD2028",
  "BIGD2029",
  "DATA2025",
  "DATA2026",
  "DATA2027",
  "DATA2028",
  "DATA2029",
  "MLAI2025",
  "MLAI2026",
  "MLAI2027",
  "MLAI2028",
  "MLAI2029"
]
```

## 사용 방법

1. 위 코드를 `prj_source/data/otp-list.json` 파일로 저장합니다.
2. 각 코드는 8자리 영문+숫자 조합입니다.
3. 입력 시 대소문자 무시 (예: `aice2025` → `AICE2025`로 인식)
4. 하이픈 자동 보정 (예: `AICE-2025` → `AICE2025`로 인식)
