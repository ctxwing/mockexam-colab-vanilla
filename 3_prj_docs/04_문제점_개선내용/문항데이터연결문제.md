# 문항데이터 연결 문제

## 발생일: 2026-04-03

---

## 문제점: Colab URL 404 오류

### 증상

**Google Colab 실습** 버튼 클릭 시 다음 오류 발생:

```
노트를 로드하는 중에 오류가 발생했습니다.
GitHub에서 이 노트북을 볼 권한이 있는지 확인하고
Colab이 GitHub API를 사용하도록 승인하세요.

https://github.com/youngjin-campaign/aice-mockexam/blob/main/AICE_01_...

Fetch for https://api.github.com/repos/youngjin-campaign/aice-mockeam/contents/...
failed: {
  "message": "Not Found",
  "status": "404"
}
```

### 원인

1. `exams.json`의 `colabUrl`이 참조하는 GitHub 저장소(`youngjin-campaign/aice-mockexam`)가 존재하지 않음
2. V1 프로젝트에서 사용하던 Colab 노트북이 실제로 생성되지 않았음
3. URL이 placeholder 형태로만 존재하고 실제 리소스 없음

### 해결 (일시적 조치)

**2026-04-03 기준**: `colabUrl`을 실제 작동하는 참고 사이트 URL로 대체

| 시험 | 기존 Colab URL (404) | 변경된 URL |
|-----|---------------------|-----------|
| AICE 1회 | `github.com/youngjin-campaign/...` | `velog.io/@junseyeon/AICE-associate-...` |
| AICE 2회 | `github.com/youngjin-campaign/...` | `blog.naver.com/yiyangse/223488690089` |
| 빅데이터 1회 | `github.com/youngjin-campaign/...` | `lab.statisticsplaybook.com/bae-exam-7/` |
| 빅데이터 2회 | `github.com/youngjin-campaign/...` | `lab.statisticsplaybook.com/bae-exam-10/` |

### 추후 해결 방안

#### 옵션 1: 실제 Colab 노트북 생성 (권장)

1. **GitHub 저장소 생성**
   ```bash
   # 예: github.com/username/aice-mockexam
   ```

2. **Colab 노트북 작성**
   - 각 시험별로 `.ipynb` 파일 생성
   - 문제 데이터, 코드, 해설 포함

3. **URL 업데이트**
   - 관리자 페이지(`admin.html`)에서 `colabUrl` 업데이트
   - 빌드 후 커밋

#### 옵션 2: 로컬 Colab 파일 사용

프로젝트 내에 Colab 노트북을 포함하고 상대 경로 사용:

```json
{
    "colabUrl": "../notebooks/AICE_01_내비게이션_주행데이터_도착시간_예측.ipynb"
}
```

#### 옵션 3: Colab 대체 서비스 사용

- **Google Colab Enterprise**: 조직용 Colab
- **DeepNote**: 협업-friendly 대안
- **Jupyter nbviewer**: 정적 노트북 뷰어

## 영향 범위

### 영향 받는 기능

1. **로비 화면**
   - [Google Colab 실습] 버튼 (현재는 참고 사이트로 연결)

2. **학습 화면**
   - Colab 연동이 필요한 경우

### 영향 받지 않는 기능

1. **정답 및 해설** (`solutionUrl`) - 정상 작동
2. **강의/참고자료** (`lectureUrl`) - 정상 작동
3. **인증 시스템** - 정상 작동
