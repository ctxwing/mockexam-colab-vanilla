# 기술 스택 및 외부 라이브러리

## 핵심 기술

| 분류 | 기술 | 버전 | 용도 |
|------|------|------|------|
| 마크업 | HTML5 | - | 페이지 구조 |
| 스타일 | CSS3 | - | 레이아웃 및 스타일링 |
| 스크립트 | JavaScript (ES6+) | - | 클라이언트 로직 |

## 외부 라이브러리

| 라이브러리 | 용도 | CDN/연결 |
|-----------|------|---------|
| Kakao JavaScript SDK | 카카오톡 공유 기능 | `https://developers.kakao.com/sdk/js/kakao.min.js` |
| highlight.js | 코드 구문 강조 | `https://cdnjs.cloudflare.com/ajax/libs/highlight.js/` |

## 개발 도구

| 도구 | 용도 |
|------|------|
| Live Server / Python http.server | 로컬 개발 서버 |
| W3C Validator | HTML/CSS 문법 검증 |
| GitHub Pages | 배포 플랫폼 |

## 폴더 구조

```
prj_source/
├── assets/
│   ├── images/     # 로고, 도서 이미지
│   └── icons/      # 파비콘 등
├── css/
│   ├── reset.css           # CSS 초기화
│   ├── variables.css       # CSS 변수 정의
│   ├── common.css          # 공통 컴포넌트
│   └── responsive.css      # 반응형 미디어 쿼리
├── js/
│   ├── common.js           # 공통 유틸리티
│   ├── device-detect.js    # 모바일/PC 감지
│   ├── storage.js          # Local Storage 래퍼
│   ├── auth.js             # 인증 로직
│   ├── lobby.js            # 로비 화면 로직
│   ├── learning.js         # 학습 화면 로직
│   └── mobile.js           # 모바일 안내 로직
├── data/
│   ├── exams.json          # 모의고사 메타데이터
│   ├── otp-list.json       # 인증 코드 목록
│   └── solutions/          # 해설 파일 (*.md)
├── pages/
│   ├── auth.html           # 인증 페이지
│   ├── lobby.html          # 로비 페이지
│   ├── learning.html       # 학습 페이지
│   ├── mobile-intro.html   # 모바일 안내
│   └── 404.html            # 404 페이지
├── public/
│   └── colab/              # Colab 노트북 파일 (*.ipynb)
└── index.html              # 메인 진입점 (리다이렉트)
```

## 설계 원칙

1. **Vanilla JS 우선**: 프레임워크 없이 순수 JavaScript
2. **데이터/로직 분리**: JSON 파일로 콘텐츠 관리
3. **클라이언트 기반**: 서버 통신 없이 Local Storage 활용
4. **정적 배포**: GitHub Pages 호스팅
