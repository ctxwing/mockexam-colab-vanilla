# 관리자 매뉴얼

Colab 기반 교육용 가상 모의고사 웹 서비스
버전: 1.0
작성일: 2025-12-23

---

## 1. 개요

본 매뉴얼은 모의고사 웹 서비스의 콘텐츠 관리 방법을 안내합니다.
모든 관리 작업은 **Google Drive 폴더**에서 수행하며, 별도의 개발 지식이 필요하지 않습니다.

### 1.1 관리 항목

| 항목 | 관리 위치 | 빈도 |
|------|----------|------|
| 모의고사 목록 | 문항관리 시트 > exams | 신규 추가 시 |
| 인증 코드 | 문항관리 시트 > auth_codes | 신판 발행 시 |
| 해설 콘텐츠 | Solutions 폴더 > *.gdoc | 수정 필요 시 |
| 답안 제출 폼 | Forms 폴더 > *.gform | 신규 추가 시 |

### 1.2 사전 준비

- Google 계정 로그인
- 관리용 GDF(Google Drive Folder) 접근 권한 확인

## 2. 인증 코드 관리

### 2.1 인증 코드 추가

1. Google Drive에서 `문항관리` 스프레드시트 열기
2. 하단의 `auth_codes` 탭 클릭
3. 마지막 행 아래에 새 행 추가
4. 입력:
   - **code**: 16자리 코드 (예: ABCD-EFGH-IJKL-MNOP)
   - **isActive**: 체크 (✓)
   - **note**: 메모 (선택, 예: "초판 1쇄")
5. 저장 (자동 저장)

**반영 시간**: 최대 24시간 (캐시 갱신 주기)
**즉시 반영 필요 시**: 개발자에게 "캐시 갱신" 요청

### 2.2 인증 코드 비활성화

1. 해당 코드 행의 `isActive` 체크 해제
2. 저장 (자동 저장)

### 2.3 인증 코드 삭제

1. 해당 행 전체 선택 (행 번호 클릭)
2. 마우스 우클릭 > "행 삭제"

⚠️ **주의**: 삭제된 코드는 복구 불가. 비활성화 권장.

## 3. 모의고사 관리

### 3.1 새 모의고사 추가

**사전 작업:**
- Solutions 폴더에 해설 문서(Google Docs) 준비
- Forms 폴더에 제출 폼(Google Forms) 준비
- Colab 노트북 URL 준비

**추가 절차:**
1. `문항관리` 시트에서 `exams` 탭 열기
2. 마지막 행 아래에 새 행 추가
3. 각 컬럼 입력:

| 컬럼 | 입력 방법 |
|------|----------|
| id | 고유 식별자 직접 입력 (예: bigdata-03) |
| title | 제목 입력 |
| description | 간단 설명 입력 |
| category | 드롭다운에서 선택 (bigdata / aice) |
| **solutionDoc** | 드롭다운에서 해설 문서 선택 |
| solutionDocId | (자동 입력) |
| solutionUrl | (자동 입력) |
| **formName** | 드롭다운에서 제출 폼 선택 |
| formUrl | (자동 입력) |
| colabUrl | Colab 노트북 URL 붙여넣기 |
| dataUrl | 실습 데이터 URL 붙여넣기 |
| order | 표시 순서 숫자 입력 |
| isActive | 체크 (✓) |

4. 저장 (자동 저장)

### 3.2 모의고사 수정

1. 해당 행 찾기
2. 수정할 셀 클릭 후 내용 변경
3. 저장 (자동 저장)

**드롭다운 항목 (solutionDoc, formName) 변경 시:**
- 드롭다운 클릭 → 새 항목 선택
- ID와 URL이 자동으로 갱신됨

### 3.3 모의고사 숨기기 (비활성화)

1. 해당 행의 `isActive` 체크 해제
2. 웹사이트 목록에서 해당 모의고사가 숨겨짐

### 3.4 표시 순서 변경

1. `order` 컬럼의 숫자 수정
2. 숫자가 작을수록 먼저 표시됨

## 4. 해설 콘텐츠 관리

### 4.1 새 해설 문서 생성

1. Google Drive에서 `Solutions` 폴더 열기
2. 새로 만들기 > Google 문서
3. 파일명 규칙: `[시험종류]_[회차]_해설`
   - 예: `빅분기_3회_해설`
4. 문서 작성 (아래 템플릿 참조)
5. 공유 설정:
   - 우측 상단 "공유" 클릭
   - "일반 액세스" > "링크가 있는 모든 사용자"
   - "뷰어" 권한 확인
   - "완료" 클릭

### 4.2 해설 문서 템플릿

```
제목: [시험명] 정답 및 해설

---

[작업형 제1유형] 데이터 전처리

Q1. [문제 제목]
문제: [문제 설명]
데이터셋: [URL]

[코드 및 해설]

---

[작업형 제2유형] 모델 구축

Q4. [문제 제목]
...

---

[작업형 제3유형] 통계 검정

Q5. [문제 제목]
...
```

### 4.3 해설 문서 수정

1. Solutions 폴더에서 해당 문서 열기
2. 내용 수정 후 저장 (자동 저장)
3. 웹사이트에 변경사항이 바로 반영됨 (iframe 방식)

## 5. 제출 폼 관리

### 5.1 새 제출 폼 생성

1. Google Drive에서 `Forms` 폴더 열기
2. 새로 만들기 > Google 설문지
3. 파일명 규칙: `[시험종류]_[회차]_제출폼`
   - 예: `빅분기_3회_제출폼`
4. 문항 추가 (아래 예시 참조)
5. 설정:
   - 응답 탭 > 스프레드시트 아이콘 > "새 스프레드시트 만들기"
   - 응답 시트가 자동 생성됨

### 5.2 빅분기 제출 폼 문항 예시

| 문항 | 유형 | 설명 |
|------|------|------|
| Q1 답안 | 단답형 | 공장 이름 3개 (쉼표 구분) |
| Q2 답안 | 단답형 | 숫자 (소수점 둘째자리) |
| Q3 답안 | 단답형 | 연도, 분기 형식 |
| Q4 결과 파일 | 파일 업로드 | result.csv |
| Q5 답안 | 단답형 | 평균, 표준편차 |
| Q6 답안 | 단답형 | 숫자 |
| Q7 답안 | 객관식 | 채택 / 기각 |
| Q8~Q10 | 단답형 | 숫자 |

### 5.3 폼 응답 확인

1. Forms 폴더에서 해당 폼 열기
2. 상단 "응답" 탭 클릭
3. 스프레드시트 아이콘 클릭 → 응답 시트에서 확인

## 7. 자주 묻는 질문 (FAQ)

### Q: 드롭다운에 새 파일이 안 보여요

**A:** 시트를 새로고침(Ctrl+R) 하거나, 시트를 닫았다가 다시 열어보세요. 드롭다운은 시트를 열 때 갱신됩니다.

### Q: 변경했는데 웹사이트에 반영이 안 돼요

**A:** 캐시 갱신 주기(24시간) 때문입니다. 즉시 반영이 필요하면 개발자에게 "재빌드" 요청하세요.

### Q: 인증 코드 형식은 어떻게 하나요?

**A:** 16자리 영숫자, 하이픈으로 4자리씩 구분 권장 (XXXX-XXXX-XXXX-XXXX). 대소문자는 구분하지 않습니다.

### Q: 해설 문서를 삭제하면 어떻게 되나요?

**A:** 웹사이트에서 해당 해설을 볼 수 없게 됩니다. 먼저 exams 시트에서 해당 모의고사를 비활성화하세요.

### Q: 파일 업로드 용량 제한이 있나요?

**A:** Google Forms 파일 업로드 제한: 파일당 10GB, 총 10GB (무료 계정 기준 Google Drive 용량 내)
